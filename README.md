# Mini Blog React

## 📌 Descripción

Mini-blog desarrollado con React + Vite + React Router. Permite visualizar un listado de elementos, ver su detalle y navegar entre páginas.

## 🎯 Nivel

Proyecto desarrollado apuntando a nivel: **Senior (100 pts)**

## 🚀 Tecnologías usadas

* React
* Vite
* React Router DOM v6
* Axios
* Context API
* PropTypes
* JSON Server (API simulada)

## 📂 Estructura

* `/` → Home
* `/items` → Listado
* `/items/:id` → Detalle
* `*` → Página 404

## ⚙️ Instalación

```bash
npm install
```

## ▶️ Ejecutar proyecto

Frontend:

```bash
npm run dev
```

Backend (API fake):

```bash
npm run server
```

## 🔍 Funcionalidades

* Navegación con React Router
* Página 404
* Búsqueda/filtro de elementos
* Botón de elemento aleatorio
* Estado global con Context API (favoritos / tema)
* Componentes reutilizables
* Uso de PropTypes
